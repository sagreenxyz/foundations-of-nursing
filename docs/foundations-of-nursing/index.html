A Comprehensive Pre-Licensure Textbook

Welcome to Foundations of Nursing, an open-access, comprehensive textbook for pre-licensure nursing students pursuing Associate Degree (ADN) or Bachelor of Science in Nursing (BSN) programs.

Select a chapter from the sidebar to begin reading. Chapters marked Draft Available contain a full first draft. Chapters marked Outline Only have a detailed outline but no prose draft yet. All remaining chapters are Coming Soon and will be added in future sprints.

All content follows APA 7th edition citation style and is written at a formal, academic level appropriate for pre-licensure nursing education. NCLEX-style review questions, case studies, learning objectives, and key terms are embedded in every completed chapter.

Quick-Start Guide

  • Click any chapter title in the sidebar to load its content.
  • Local file viewing: Because browsers block cross-origin requests when opening file:// URLs, a local web server is required to load chapter content. Run:
    python3 -m http.server 8080
    then open http://localhost:8080/docs/.
  • GitHub Pages: All chapters load automatically when the site is deployed at https://<user>.github.io/foundations-of-nursing/.
