Welcome to Clinical Pharmacology for Nurses, an open-access pharmacology textbook written for pre-licensure nursing students in ADN and BSN programs. The table of contents is synthesized from three leading nursing pharmacology references: Pharmacology and the Nursing Process (Lilley, Collins, & Snyder, 11th ed.), Lehne's Pharmacology for Nursing Care (Burchum & Rosenthal, 11th ed.), and Focus on Nursing Pharmacology (Tucker [Karch], 9th ed.).

Select a chapter from the sidebar to begin reading. Chapters marked Draft Available contain a full first draft. Chapters marked Outline Only have a detailed outline but no prose draft yet. All remaining chapters are Coming Soon and will be added in future sprints.

Content follows APA 7th edition citation style and is written at a formal, academic level appropriate for undergraduate nursing education. NCLEX-style review questions, clinical case studies, learning objectives, key terms, and nursing process applications are embedded in every completed chapter.

Quick-Start Guide

  • Click any chapter title in the sidebar to load its content.
  • Local file viewing: Because browsers block cross-origin requests when opening file:// URLs, a local web server is required to load chapter content. Run:
    python3 -m http.server 8080
    then open http://localhost:8080/docs/pharmacology/.
  • GitHub Pages: All chapters load automatically when the site is deployed at https://<user>.github.io/foundations-of-nursing/pharmacology/.
